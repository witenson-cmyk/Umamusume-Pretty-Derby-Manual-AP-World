# Umamusume Pretty Derby Manual AP World
Manual AP World for the Mobile and PC game Umamusume: Pretty Derby

* Starts with 1 Trainee.
* You are allowed to use any Rare Support Cards as well as any Borrowed Card
* Goal is to complete career goals with 3 different Trainees and Win the Trailmaker Championship
  * Trailmaker Career goals Count
* Intended for a short 3-5 hour Sync

## What is Randomized:
* Access to Support Cards
  * Number of each support card type is set to your own liking, so edit the number of Progressive Support Card items to your preference.
* Access to Legacies
  * Before Progressive Legacies is unlocked, you have to use one of the starting G rank legacies, or whichever lowest legacies you have if you got rid of them. Each Progressvie Legacy is another legacy slot.
* Access to Trainees
* Access to Scenarios
  * Each Progressvie Scenario unlocks the next scenario in order, Unity Cup and then Trackblazer
* Access to Clocks
* Ability to Learn Skills
* Ability to run in Non Goal Races
  * For Trailmaker, can run until you reach the required point total.
 
## What to Edit Before Playing
* Which Trainees you want to use
  * Make sure to change the item name and location name for career goals completion to the name of the Trainee you want to use
* Use https://manualforarchipelago.github.io/ManualBuilder to edit AP World
